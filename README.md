# Five Nights at Chesse

A tiny, browser-based Five Nights At Freddy's style mini-game. It's intentionally simple so you can run it locally or extend it.

How to run

- Open index.html in your browser (double-click or serve with a static server).

Rules

- Survive 60 seconds to win.
- Viewing cameras and closing doors consumes power.
- If the animatronic reaches your office while the doors are open, it's game over.

Extending

- Add images/sounds in an `assets/` folder and update `index.html`/`game.js`.
- Tweak parameters in `game.js` for difficulty.
