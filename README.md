# Five Nights at Chesse

A tiny, browser-based Five Nights At Freddy's style mini-game. I've added simple SVG textures for background, cameras, doors, and the animatronic face.

How to run

- Open index.html in your browser (double-click or serve with a static server).

Assets

- assets/texture_bg.svg — background texture
- assets/cam_frame.svg — camera thumbnail frame texture
- assets/door_texture.svg — door texture
- assets/static.svg — camera static used when power is out
- assets/anim_face.svg — animatronic face shown on camera

Extending

- Replace the SVGs with PNGs or higher-detail assets if you want richer visuals. Update CSS/HTML accordingly.
